Run GAMDL Web on your computer in minutes

Installation Steps
Prerequisites
  • Python 3.10 or higher
  • FFmpeg (for audio processing)
  • Apple Music subscription and cookies
Quick Installation (macOS/Linux)
# Clone the repository
git clone https://github.com/nyatanvn/gamdl.git
cd gamdl

# Run the startup script
./start.sh
Manual Installation
# Install dependencies
pip install -r requirements.txt
pip install -r web_requirements.txt

# Start the web app
python web_app.py
Get Apple Music Cookies

Use one of these browser extensions to export your Apple Music cookies:

  • Firefox: Export Cookies
  • Chrome: Get cookies.txt
